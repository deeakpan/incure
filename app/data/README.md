# Natural Earth GeoJSON Data

## Download Instructions

1. Go to: https://www.naturalearthdata.com/downloads/110m-cultural-vectors/
2. Download the "Admin 0 – Countries" file
3. Extract the ZIP file
4. Find `ne_110m_admin_0_countries.geojson`
5. Copy it to this folder (`app/data/`)

OR use this direct link:
https://raw.githubusercontent.com/holtzy/D3-graph-gallery/master/DATA/world.geojson

## File Info

- **File name**: `ne_110m_admin_0_countries.geojson`
- **Size**: ~2MB
- **Contains**: 195 country polygons
- **Format**: GeoJSON
- **License**: Public Domain
